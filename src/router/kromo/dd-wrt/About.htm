About DD-WRT's distributions

The Mini
Includes the same features of the standard distribution, with the exception of chillispot, nocat, rflow, kaid, CIFS client, SNMP, IPv6, and MMC/SD card support.
The Standard
The (std) includes all features, with the exception of VOIP.
The Standard Nokaid
The (std-nokaid) includes the same features of the standard distribution, with kaid removed to free some flash space.
The VOIP
This includes all features, with kaid removed to make room for Milkfish.
The VPN
It includes OpenVPN but does not include IPv6, CIFS client, or kaid.
The Micro
This is a stripped down version designed for the WRT54G v5 and other 2MB router models.
