MeetmeMute

Synopsis

Raised when a MeetMe user is muted.

Syntax

Event: MeetmeMute
Meetme: <value>
Usernum: <value>
Status: <value>
Channel: <value>
Uniqueid: <value>
Meetme: <value>
Usernum: <value>
Arguments
  • Meetme - The identifier for the MeetMe conference.
  • Usernum - The identifier of the MeetMe user who joined.
