ConfbridgeJoin

Synopsis

Raised when a channel joins a Confbridge conference.

Syntax

Event: ConfbridgeJoin
Conference: <value>
Channel: <value>
Uniqueid: <value>
Conference: <value>
CallerIDnum: <value>
CallerIDname: <value>
Arguments
  • Conference - The name of the Confbridge conference.
  • Channel
  • Uniqueid
  • Conference
  • CallerIDnum
  • CallerIDname
