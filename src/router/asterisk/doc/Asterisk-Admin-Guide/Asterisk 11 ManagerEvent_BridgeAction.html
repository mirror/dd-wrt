BridgeAction

Synopsis

Raised when a bridge is successfully created due to a manager action.

Syntax

Event: BridgeAction
Response: <value>
Channel1: <value>
Channel2: <value>
Arguments
  • Response
    • Success
    • Failed
  • Channel1
  • Channel2
