REALTIME_DESTROY()

Synopsis

RealTime Destroy Function.

Description

This function acts in the same way as REALTIME(....) does, except that it destroys the matched record in the RT engine.

Syntax

REALTIME_DESTROY(family,fieldmatch,matchvalue,delim1,delim2)
Arguments
  • family
  • fieldmatch
  • matchvalue
  • delim1
  • delim2
