MeetmeTalking

Synopsis

Raised when a MeetMe user begins or ends talking.

Syntax

Event: MeetmeTalking
Meetme: <value>
Usernum: <value>
Status: <value>
Channel: <value>
Uniqueid: <value>
Meetme: <value>
Usernum: <value>
Arguments
  • Meetme - The identifier for the MeetMe conference.
  • Usernum - The identifier of the MeetMe user who joined.
  • Status
    • on
    • off
  • Channel
  • Uniqueid
  • Meetme
  • Usernum
